## 🚀 部署指南

### GitHub Pages 部署

1. **Fork 项目**
```bash
git clone https://github.com/xuanxuan205/Classic-cloud.git
cd Classic-cloud
```

2. **推送到您的仓库**
```bash
git remote set-url origin https://github.com/YOUR_USERNAME/Classic-cloud.git
git push origin main
```

3. **启用 GitHub Pages**
- 进入仓库设置
- 找到 "Pages" 选项
- 选择 "Deploy from a branch"
- 选择 "main" 分支
- 保存设置

### 自定义域名

1. **添加 CNAME 文件**
在仓库根目录创建 `CNAME` 文件，内容为您的域名。

2. **配置 DNS**
在您的域名服务商添加 CNAME 记录：
```
类型: CNAME
名称: @ (或 www)
值: YOUR_USERNAME.github.io
```

## 🔧 自定义配置

### 添加新功能

1. 在 `assets/js/main.js` 中添加新的 JavaScript 功能
2. 在 `assets/css/style.css` 中添加对应样式
3. 更新 `index.html` 中的页面结构
